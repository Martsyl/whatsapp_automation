🔑 Getting your Access Token
1

Go to business.facebook.com → Business Settings

2

Click Users → System Users → Add

3

Name it "WhatsApp Bot" and set role to Admin

4

Click Add Assets → WhatsApp Accounts → select your account → Full Control

5

Click Generate Token → select your app → enable whatsapp_business_messaging → copy token

📱 Getting Phone Number ID & WABA ID
1

Go to developers.facebook.com → My Apps

2

Select your app → WhatsApp → API Setup

3

You'll see Phone Number ID and WhatsApp Business Account ID listed there
